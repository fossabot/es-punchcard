# es-punchcard


##Description
Make punchcard charts based on Elasticsearch histogram aggregations

##Documentation

##Installation
###Git clone
```
git clone https://github.com/maziyarpanahi/es-punchcard.git
```
###Bower
```
bower install es-punchcard
```
